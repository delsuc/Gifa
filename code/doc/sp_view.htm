The Specific Documentation of Gifa V4.4

The specific documentation is composed of six parts. All the different documentations are available on-line by web consulting or off-line by the way of downloading the pdf file corresponding to your need.

Here is a brief description of the differents parts :

The Macro Programming language. This documentation refere to the way of personalize your own gifa software, how to write some macros and to use it simply. Here is the contents of the documentation :

  • Introduction
  • Basics of macro programming
  • Variables
  • Contexts (Internal variables)
  • Evaluated expressions
  • Control Structures
  • Support for data-bases
  • Environment
  • Parameters passing and interactivity
  • Text file input/output
  • Graphic programming and graphic interactivity
  • Graphic user interface
  • Customizing the interface
  • Miscellaneous commands, not specific but usefull
  • Exemples

The assignment Module. This documentation deal with the way to use the assignment module of gifa, how to integrate some tips on dynamic works etc... It's composed of the following chapters :

  • Setting-up
  • Menus Descriptions
  • Integration
  • Dynamic
  • Typical Work
  • Macro Programming
  • Caveat.

The Maximum Entropy Processing. Everything you always want to know about processing with the Maximum Entropy.

The Linear Prediction Processing. A complete presentation of the kind of processing.

The DOSY Processing. A complete on line manual based on a new spectroscopy feature. It's composed of :

  • Introduction.
  • Theory.
  • Implementation.
  • Usage.
  • DOSY Menu.
  • Related Contexts.

The NOESY Build-up curve Processing. A real specific documentation dealing with the building-up curves problems, composed of faur parts :

  • The Build-up curves Processing ni Gifa
  • Theory
  • Practising the Build-up curves processing
  • Somes references.
