JCONS 

JCONS J1 {J2) k

Permits to set up the J (J1 and J2) constant for J-deconvolution; 
the J entered should be half the J of the coupling you want to 
deconvoluate. k is either A or P if you want Antiphase or Inphase J-
deconvolution.


 see also : FILTER JMULT

JMULT 

JMULT J1 {J2) k
Set up the J (J1 and J2) constant for J-deconvolution; the J entered 
should be half the J of the coupling you want to deconvoluate. k is 
either A or P if you want Antiphase or Inphase J-deconvolution. The 
data are actually multiplied by the filter function. Used mostly to 
check the effect of the filter one the Data.
DO NOT use JMULT before launching a J-deconvolution Maximum Entropy 
run.


 see also : EM FILTER GM JCONS SIN SQSIN TM

JOIN 

JOIN filename 

Connects the cache memory system to the file `filename' in standard 
format ( created with WRITE or NEWFILEC). The file can then accessed 
randomly with GETC and PUTC commands

related contexts : $C_ABSMAX $C_DIM $C_FREQ $C_FREQ1 $C_FREQ2 $C_FREQ3
